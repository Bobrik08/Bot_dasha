## Описание бота

Бот-модератор для Telegram-чата.  
Основная идея:

- хранить список пользователей, которых нельзя пускать в чат (чёрный список);
- давать админам команды для управления этим списком;
- уметь по запросу проверять участников чата и показывать, кого надо заблокировать.

сейчас реализована учебная версия без настоящего бана — логика работает в памяти процесса.

## Локальный запуск

1. Клонировать репозиторий и перейти в папку проекта:

   git clone https://github.com/Bobrik08/Bot_dasha.git
   cd Bot_dasha
   
   python3 -m venv .venv
   source .venv/bin/activate
   
   pip install -r requirements.txt

Основные команды бота

Команды сейчас реализованы через aiogram-хендлеры.

/start
    •    Приветственное сообщение.
    •    Кратко объясняет, что делает бот.
    •    Показывает простую клавиатуру с часто используемыми командами (/stats, /force_check).

Доступна всем пользователям.

## Команды бота

- `/start` — приветствие, краткое описание бота и вывод простого меню с кнопками.
- `/adduser ID` — добавить пользователя с указанным ID в чёрный список
  (можно передать ID аргументом или ответить командой на сообщение пользователя).
- `/deluser ID` — убрать пользователя с указанным ID из чёрного списка.
- `/stats` — показать статистику: сколько пользователей в чёрном списке, сколько было действий, последнее действие.
- `/force_check` — учебная принудительная проверка: вывести список ID из чёрного списка, которых «надо забанить» в чате.

Все команды, кроме `/start`, доступны только администраторам (их Telegram ID задаются в переменной `ADMIN_IDS` в `.env`).
