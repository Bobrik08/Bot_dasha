1) Как только сели делать проект, получим актуальные данные: 
  git checkout main
  git pull origin main

2) Создаем новую ветку и там работаем

3)коммитим изменения

  git add все фйалики которые менялись
  git commit -m "что изменил тут"
  git push origin <название ветки>
